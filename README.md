# Endless Waves Survival

Browser roguelike survivor game built with PixiJS and TypeScript.

**[Play online](https://iaanzhao.github.io/endlesswavesurvival/)**

## Play locally

```bash
npm install
npm run dev
```

Open http://localhost:5174/

## Controls

| Input | Action |
|-------|--------|
| WASD / arrows | Move |
| Space | Dash |
| Esc / P | Pause |

Skills and basic attacks fire automatically. Level up to pick upgrades. Collect gold for the permanent shop between runs.

## Build

```bash
npm run build
npm run preview
```
